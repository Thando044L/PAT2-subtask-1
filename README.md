# What Morse code is 
Morse code is a method of encoding text characters using standardized sequences of two different signal durations called dots and dashes, dits and dahs. Each letter, number, and punctuation mark is represented by a unique combination of dots and dashes. It was designed for electrical telegraph systems and can be transmitted as sound, light, or radio signals. Spaces are used to separate letters and words.
# the history of Morse code 
Morse code was developed in the 1830s–1840s by Samuel Morse and Alfred Vail for use with the electric telegraph. The telegraph could only send pulses of electricity, so they needed a simple code to represent the alphabet. Morse code allowed messages to be sent quickly over long distances, revolutionizing 19th-century communication. It became the global standard for telegraphy and was later critical for maritime and aviation communication, especially the SOS distress signal
## How the Morse Code System Works: Dots and Dashes
1.Basic signals: A dot is the shortest signal. A dashor is three times as long as a dot. 
2.Timing rules:
Intra-character spacing: The space between dots and dashes within the same letter is one dot duration.
Inter-character spacing: The space between letters is three dot durations.
Word spacing: The space between words is seven dot durations
